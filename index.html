Navegadores web

Un navegador web (en ingles, web browser) es un software, aplocación o programa que permite el acceso a la web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incluidos

Ejemplos de navegadores web

Safari: safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e IOS.

Mozilla Firefox es un navegador web libre y de código abierto desarollado para Linux, Android, IOS, macOS y Microsoft Windows coordinado por la Coordinación Mozilla y la Fundación Mozilla.

Google Chrome: Google Chrome es un navegador web de código cerrado desarollado por Google, aun que derivado de proyectos de código abierto.
